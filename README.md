# Popup Blueprints (Browser Mod)

Raccolta di blueprint per Home Assistant con popup in stile app,
pensati per dashboard tablet e wall panel tramite Browser Mod.

Questa repository include un piccolo **custom component** che
sincronizza automaticamente i blueprint nella cartella ufficiale di Home Assistant,
gestendo gli aggiornamenti tramite HACS.
